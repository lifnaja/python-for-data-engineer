# Basic Operators

## Arithmetic Operators

ใน python มีตัวดำเนินการบวก ลบ คูณ และหาร สามารถใช้งานกับตัวเลขได้

```py linenums="1"
number = 1 + 2 * 3 / 4.0
print(number)
```


### Another operator
- modulo operator (%)

```py linenums="1"
number = 9 % 3
print(number)

number = 7 % 3
print(number)
```

- power operator (%)

```py linenums="1"
squared = 7 ** 2
cubed = 2 ** 3
print(squared)
print(cubed)
```

- floor division (//)
```py linenums="1"
num = 7 // 2
print(num)
```


## String Operators

- Concatenation

```py linenums="1"
str1 = "Hello"
str2 = "World"
result = str1 + " " + str2
print(result)
# f-string
```

